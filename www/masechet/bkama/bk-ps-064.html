1)THE SOURCE FOR PAYING KEFEL, FOUR AND FIVE

(a)Question (Tana d'Vei Chizkiyah): The Torah should have said only 'ox' and 'theft', and we would know that Kefel applies to everything!

(b)Answer: If the only example (of a stolen object) was 'ox', we would include only animals that are offered on the Mizbe'ach!

1.This would include a Seh. However, the verse also says "Seh"! We must say that "theft..." includes everything! (The Gemara proceeds to explain this.)

2.Question: It would have sufficed to write 'ox, Seh, theft'!

3.Answer: If so, we would learn only animals whose firstborn have Kedushah.

i.This would include a donkey. However, the verse explicitly says "donkey"! We must say that "theft..." includes everything!

4.Question: It would have sufficed to write 'ox, donkey, Seh, theft'!

5.Answer: If so, we would include only living things.

i.The verse says also "living"! We must say that ("theft...") includes everything!

(c)Question: Why did Tana d'Vei Chizkiyah say 'had the Torah said only 'ox' and 'theft''? "Theft" is written before "ox"!

1.Suggestion: The Tana means 'had the Torah written 'ox... theft...'

2.Rejection: If so, 'ox' would be a Prat, and 'theft' a Klal. From a Prat u'Chlal we learn everything!

(d)Answer: Rather, the Tana means 'had the Torah written only 'theft... ox...'

(e)Question: If so, why would we think that all is included?

1.'Theft' would be a Klal, and 'ox' would be a Prat. From a Klal u'Frat we learn only the Prat, an ox!

(f)Answer (Rava): The Tana relies on 'living', a second Klal. There is a Klal u'Frat u'Chlal.

(g)Question: The second Klal (includes only living things. This) is unlike the first Klal!

(h)Answer: He holds like Tana d'Vei R. Yishmael, who expounds such a Klal u'Frat u'Chlal. The Tana asks as follows:

1.Question: Why do we need "Im Himatzei Simatzei?" We could learn everything from (the Klal u'Frat u'Chlal) 'ox, theft, living'!

2.Answer: If the only Prat was 'ox', we would include only animals that are offered on the Mizbe'ach.

3.Question: This would include a Seh. However, the Torah explicitly wrote "Seh"!

i.Rather, we must say that "theft..." comes to include everything (through the method of Klalim and Peratim);

ii.(Summation of question): It would have sufficed to write 'theft, ox, Seh, living'!

4.Answer: If so, we would have included only animals whose firstborn have Kedushah.

5.Question: That would include a donkey. However, the Torah explicitly wrote "donkey"!

i.We must say that "theft..." includes everything. It would have sufficed to write 'theft, ox, Seh, donkey, living'!

6.Answer: If so, we would include only living things.

7.Question: The Torah explicitly writes "living" (which includes all living things)!

i.We must say that "theft..." includes everything. Why did the Torah need to write "Im Himatzei Simatzei"?

(i)Question: If this was the question, how can we answer it?!

(j)Answer: We asserted that we could have included everything from 'theft, ox, Seh, donkey, living.' This is wrong. We cannot include more than the latter Klal, which is "living"!

1.Question: What do we learn from the Klal u'Frat u'Chlal?

i.We cannot learn everything, for the latter Klal is "living", which can include only living things!

2.(Summation of answer): This is why we need "Im Himatzei Simatzei" (to include everything).

2)ADJACENT KELALIM

(a)Question: The Peratim are not between the Klalim "Himatzei" and "Simatzei"!

(b)Answer #1 (Ravina): In such a case, we consider it as if the Peratim were in the middle, to expound like a Klal u'Frat u'Chlal.

1.Question: We put "ox" between "Himatzei" and "Simatzei". What does this include?

i.It need not include living things. We already know this from "living"!

2.Answer: Rather, it includes inanimate things.

i.We expound: the Prat is movable and has intrinsic value. We include all such things.

3.Question: We put "donkey" between "Himatzei" and "Simatzei." What does this include?

i.It need not include inanimate things. We already know them from "ox"!

4.Answer: Rather, it includes specific things (Rashi - that have Simanim; Tosfos - they have a measure, or are whole).

5.Objection: If so, what do we learn from "Seh"?

(c)Answer #2: Rather, we expound it through Ribuyim and Mi'utim, like Tana d'Vei R. Yishmael.

1.(Beraisa - Tana d'Vei R. Yishmael): We do not expound "in water... in water" through the method of Klalim and Peratim. Rather, it is a Ribuy u'Mi'ut u'Ribuy.

(d)Question: If it is a Ribuy u'Mi'ut u'Ribuy, everything is included. Why were the Peratim written?

(e)Answer: One excludes land, one excludes slaves, and one excludes documents;

1."Theft" and "living" teach like Rav, that the thief must resuscitate what he stole (restore it to the value at the time he stole it).

3)WHAT DO WE LEARN FROM IM HIMATZEI SIMATZEI?

(a)Question: According to Beraisa #2 (one verse discusses a thief, and the other discusses one who claims that the deposit was stolen), what do we learn from "Im Himatzei Simatzei"?

(b)Answer: It teaches Rava bar Ahilai's law;

1.(Rava bar Ahilai citing Rav): If \is this how you want?\one admits to a fine, and witnesses came later, he is exempt. He learns from "Im Himatzei Simatzei";

i.If Himatzei (he will be found) through witnesses, judges Simatzei (will find him) liable. This excludes one who incriminates himself (even if witnesses come later).

(c)Question: According to Beraisa #1 (both verses discuss one who claims that the deposit was stolen), from where do we know this?

(d)Answer: "Whom the judges will convict" excludes one who incriminates himself.

(e)Question: What does the Tana of Beraisa #2 learn from this verse?

(f)Answer: It exempts one who admits to a fine. (This allows him to learn from "Im Himatzei..." that even if witnesses come later, he is exempt.)

1.The Tana of Beraisa #1 only has one verse exempting one who admits to a fine. He holds that if witnesses come later, he is liable.

(g)Question: According to Beraisa #2, we said that "Im Himatzei Simatzei" teaches like Rava bar Ahilai. What do the Peratim teach?

(h)Answer: This is like Tana d'Vei R. Yishmael taught:

1.(Beraisa - Tana d'Vei R. Yishmael): An entire Parshah can be repeated for the one Chidush. (Here, Im Himatzei Simatzei exempts one who admits to a fine. We did not need the Peratim, for we can learn from the coming Parshah (Im Lo Yimatzei ha'Ganav, which discusses one who claims that it was stolen, and also lists the Peratim).)

(i)Suggestion: An actual thief should pay Kefel only if he swore!

(j)Rejection (Beraisa - R. Yakov): "He will pay Kefel" - without an oath.

1.Suggestion: Perhaps this is only if he swore!

2.Rejection: It was not so.

3.Question: What does this mean?

4.Answer (Abaye): The Torah did not need to write that a thief pays Kefel. We could have learned from a Kal va'Chomer:

i.One who (falsely) claims that the deposit was stolen pays Kefel, even though the object came to him in a permitted way. A thief took the object in a forbidden way, all the more so he pays Kefel!

ii.The Torah write that a thief "pays two" to teach that this is even if he did not swear.
